Projects

makeswordclouds makeswordclouds is a Reddit bot that views the hottest 100 links of /r/all and if the post has 50 comments or more, it will post a pleasing word cloud to the thread itself. This bot is trivial only, but it does provide a rough "sum" of what the thread is about.
LinkFixerBot LinkFixerBot is a simple Reddit bot that I made to allow easier access to subreddits for mobile users. Whenever a Reddit user posts a "broken" link to a subreddit, the bot will reply with the proper link.
dJRequest dJRequest is a Java application designed to play a song on the Worlds Collide Network Teamspeak. The client has a simple queue that has a list of song entries that users input in when they want to hear music. All that is required is a YouTube link to the song and the song's duration!
JoinMessages JoinMessages is a simple Bukkit plugin for the game Minecraft that allows you to setup easy custom join / leave messages when a player join or leaves the game.
Apokalypse Apokalypse is a hunger games style mini-game for the Minecraft implementation Bukkit. It is currently an early work in progress, but it is progressing rapidly.
